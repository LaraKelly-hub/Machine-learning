# projeto-tesla-regressao

# 📈 Modelo de Regressão – Ações da Tesla (TSLA)
📈 Previsão de Valores das Ações da Tesla com Machine Learning.


📌 Objetivo do Projeto
Este projeto tem como objetivo desenvolver um modelo de Inteligência Artificial capaz de prever os valores das ações da empresa Tesla, auxiliando na análise do mercado financeiro.
A partir desse modelo, é possível apoiar a tomada de decisões mais eficientes, considerando que o mercado de ações apresenta alta volatilidade e sofre constantes impactos econômicos e globais.


🛠️ Ferramentas e Tecnologias Utilizadas

Python

Bibliotecas:

Pandas

NumPy

Matplotlib

Scikit-learn

Jupyter Notebook

🔄 Etapas do Projeto

Coleta de dados:

Os dados utilizados foram obtidos a partir da plataforma Kaggle.

Tratamento e preparação dos dados:

Realizado com Python, incluindo limpeza, organização e preparação das variáveis para o modelo.

Análise exploratória e visualização:

As visualizações foram desenvolvidas com a biblioteca Matplotlib, permitindo identificar padrões, tendências e oscilações no mercado.

Treinamento e avaliação do modelo:

Aplicação de algoritmos de aprendizado supervisionado para a construção de modelos de machine learning voltados à previsão dos valores das ações..

📊 Resultados Obtidos

A análise revelou picos significativos no volume de negociações, que coincidem com eventos globais de grande impacto econômico.

Ano de maior pico de negociação: 2020
Esses resultados reforçam a forte relação entre o mercado financeiro e fatores externos, como crises globais e mudanças econômicas.

📈 Desempenho do Modelo

Métricas de Treinamento:

MAE: 0.05

RMSE: 0.27

R²: 1.00

Métricas de Teste:

MAE: 0.10

RMSE: 0.27

R²: 1.00

As métricas indicam boa eficiência do modelo, demonstrando boa capacidade de generalização, mesmo diante das oscilações naturais do mercado financeiro.

**🚀Conclusão**

O modelo apresentou desempenho consistente e resultados satisfatórios, mostrando-se uma ferramenta promissora para análises preditivas no mercado de ações.
Como próximos passos, o projeto pode ser expandido com a inclusão de novas variáveis econômicas, diferentes algoritmos e testes com outras empresas do setor financeiro.
